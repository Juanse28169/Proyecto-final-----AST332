Proyecto-final---AST332 Clasificación de Eventos Sísmicos: Detección de Terremotos y Explosiones mediante Machine Learning 

Descripción: Este proyecto utiliza técnicas de Machine Learning para clasificar eventos sísmicos en dos clases principales: Terremoto (Earthquake) (clase 0, mayoritaria) o No-Terremoto (No-Earthquake) (clase 1, minoritaria).

El código entrena un clasificador Random Forest utilizando características físicas como la Profundidad, la Ubicación (Lat/Lon) y la Magnitud, con el objetivo de distinguir entre sismos naturales y eventos artificiales o no tectónicos.
